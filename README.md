# 🏏 Cricbuzz Live Match Analysis Dashboard (Streamlit)

## 🚀 Project Overview
Developed an interactive cricket analytics dashboard using **Streamlit, Python, and MySQL**, integrating real-time and historical match data to generate insights on teams, venues, and match trends.

---

## 📌 Introduction
This project provides a **live and historical analysis of cricket matches**, inspired by Cricbuzz data.

The dashboard aggregates data from multiple sources, including:
- Matches  
- Teams  
- Series  
- Venues  

It allows users to explore:
- Live matches  
- Upcoming matches  
- Historical trends

---

## 🎯 Objective
- Build a real-time cricket analytics dashboard  
- Provide insights into match trends and team performance  
- Enable interactive data exploration  
- Track live, upcoming, and recent matches  

---

## ⚙️ Tech Stack

- **Python (Pandas, Plotly)**  
- **MySQL** (Data Cleaning & EDA)  
- **Streamlit** (Dashboard UI)  
- **Rapid API** (Data Source)  

---

## 🔄 Methodology

### 📥 Data Collection
- Data fetched using **Rapid API (Cricbuzz)**  
- Structured into datasets:
  - Matches  
  - Teams  
  - Series  
  - Venues  

### 🧹 Data Preprocessing
- Handled missing values  
- Standardized column formats  
- Converted dates into datetime format  

### 🗄️ Data Processing (MySQL + Python)
- Performed KPI calculations:
  - Total Matches  
  - Total Teams  
  - Total Series  
  - Total Venues  
- Identified:
  - Most played venues  
  - Match categories (Live / Upcoming / Recent)
